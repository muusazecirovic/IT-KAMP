Kaskadni stilovi

Postoje 3 vrste primene kaskadnih stilova na html dokumentima

  1. Direktna primena kaskadnih stilova kao atribut nakon taga koji želimo da menjamo
  2. Unutar head taga imamo style tag unutar kog vršimo primenu CSS-a
  3. Eksterni CSS karakteriše novi fajl (npr. style.css )koji će biti povezan sa našim html fajlom
